RL incentivizes LLM reasoning via spectrum space rewiring

AUTHORS

Zhilong Zhang, Hongli Yu, Huan-ang Gao, Hanlin Wu, Hao Zhou

AFFILIATIONS

GenSI, Institute for AI Industry Research (AIR), Tsinghua University

PUBLISHED

February 12, 2026

READ TIME

10 min

TL;DR

  • RL incentivizes LLM reasoning via Rewiring, Not Re-invention – RL doesn't teach models new features from scratch; it associate latent "atomic skills" by rewiring the interaction between singular components of base model.
  • Structure of Reasoning: The RL-incentivized reasoning is physically stored in the principal SVD subspace of the base model. Through extensive experiments on state-of-the-art reasoning LLMs, we show that RL-updated constrained to this narrow subspace (often just 1% of the rank) are sufficient to recover 100% of the reasoning capability.
  • Cross-Domain Generalization: By isolating these spectral updates, we can merge math-reasoning capability into coding models, and create "super models" that outperform domain-specific experts in both fields. This suggests that reasoning in SVD space is a universal signature shared across different domains, and it induces cross-domain generalization.

Content

Introduction: What is Reinforcement Learning Truly Doing in the parameter space?

Reinforcement Learning (RL) is pushing the boundaries of Large Language Models (LLMs), enabling them to achieve gold-medal performance in IMO and dominate competitive programming. Yet, the underlying mechanism of RL remains a "black box," leading to a series of fragmented and seemingly contradictory observations. For examples:

  • Is RL teaching new capabilities or merely eliciting latent ones from the base model [1]?
  • Some studies suggest RL updates are sparse and low-rank [2], while others observe that RL leads to more diverse activation pathways inside the network [3].

Our goal is to build a fundamental understanding of how RL-finetuning incentivizes the reasoning ability of LLMs. By analyzing the parameter updates of RL, we gain a transformative perspective that allows us to understand how RL incentivizes reasoning abilities, explain existing observations consistently, and unlock new potentials for RL-based reasoning models—such as cross-domain model merging.

Insights and Takeaways

Our key insight is that, if RL is primarily incentivizing the inherent reasoning capabilities of the base model, then the effective parameter updates for reasoning must reside within the geometric manifold defined by the base model itself. Essentially, RL is not a "re-invention" but a "re-wiring" along the pre-existing geometric tracks of the pre-trained weights.

Our theoretical and empirical analyses highlight three key points:

  1. Physical Manifestation: The RL-incentivized reasoning is physically stored in the Singular Value Decomposition (SVD) space of the base model. Intuitively, the singular component could be view as "atomic skills" learned by the base model, and RL-incentivized reasoning is associating existing skills to solve complex reasoning problem.
  2. Spectrum Space Rewiring: We demonstrate that parameter updates constrained strictly within this principal SVD subspace are sufficient to recover the entirety of the reasoning capabilities activated by RL. It reveals that RL does not learn new features from scratch. Instead, RL manipulates the interaction between existing features, i.e., the singular component of base model.
  3. Cross-Domain Generalization: Perhaps the most striking evidence is the composability of these updates. We found that a reasoning update learned in the spectral space of a mathematics-focused model can be merged into a coding-focused model. The result is a "super-model" that exhibits improved performance in both math and coding. This suggests that reasoning in SVD space is a universal signature shared across different domains, and it induces cross-domain generalization.

Results

To validate our insights, we conducted extensive evaluations using state-of-the-art (SOTA) open-sourece reasoning models, including: DeepscaleR [4], Polaris [5], Olmo-3 [6]. They ranged from different model families (DeepSeek-Distill-Qwen Series, Qwen3, OLMo3) with varying model sizes (1.5B,4B,7B,32B).

Our primary benchmark was the AIME 2024 dataset, which requires deep, multi-step logical synthesis. We projected the RL model to the base model using the above algorithm, and compared the performance with or without projection. We project weight matrixs of both Attention and MLP layers. The evaluation followed recommanded configuration of each reasoning model.

Why SVD? The "Rewiring" Perspective

When a representation xxx flows through a layer, the base model uses its pre-trained singular vectors uiu_iui​ to "read" the input, and singular vectors viv_ivi​ to output the information:

   ybase=W0x=∑i=1rσiuiviTx=∑iσi(viTx)uiy_{base} = W_0x = \sum_{i=1}^r \sigma_i u_i v_i^Tx = \sum_{i} \sigma_i (v_i^T x) u_iybase​=W0​x=∑i=1r​σi​ui​viT​x=∑i​σi​(viT​x)ui​

Here σi(viTx)\sigma_i (v_i^T x)σi​(viT​x)is a scaler, and thus the output is a linear combination of uiu_iui​.Specifically, the coefficient of each uiu_iui​ is triggered strictly by its corresponding viv_ivi​. These are orthogonal spectrum of latent skills.

The post-RL model can be expressed as:W1∗=U(Σ+M)VTW_1^* = U (\Sigma + M) V^TW1∗​=U(Σ+M)VT,where MMM describes how pre-trained "atomic skills" are rewired. Specifically, the output feature uiu_iui​ is now determined by a multitude of input directions:

   yrewired=U(Σ+M)VTx=∑i[(σi+Mii)(viTx)⏟Rescaling+∑j≠iMij(vjTx)⏟Rewiring]uiy_{rewired} = U (\Sigma + M) V^Tx = \sum_{i} \left[ \underbrace{(\sigma_i + M_{ii}) (v_i^T x)}_{\text{Rescaling}} + \underbrace{\sum_{j \neq i} M_{i\textcolor{red}{j}} (v_{\textcolor{red}{j}}^T x)}_{\text{Rewiring}} \right] u_iyrewired​=U(Σ+M)VTx=∑i​​Rescaling(σi​+Mii​)(viT​x)​​+Rewiringj=i∑​Mij​(vjT​x)​​​ui​

As we can see, the diagonal part (Mi,iM_{i,i}Mi,i​) is rescaling the original latent skills (σi(viTx)ui\sigma_i (v_i^T x) u_iσi​(viT​x)ui​). More importantly, the off-diagonal terms (Mi,jM_{i,j}Mi,j​) create new connections between viv_ivi​ and uiu_iui​. This is the physical essence of reasoning: the coefficient of output skill uiu_iui​ is now determined by multiple input signals vjv_jvj​, i≠ji \neq ji=j. This shift from "point-to-point" signals to "many-to-one" logic synthesis directly accounts for the increased capacity of solving complex and multi-step reasoning task throught composition of learned skills.

An Intuitive Example

From a cognitive perspective, the process of Rewiring is equivalent to the concept of "Association" in human intelligence.

The Problem: Suppose the model is given a prompt: "Find the area of a triangle where the lengths of two sides are a=5a=5a=5 and b=8b=8b=8, and the included angle CCC is 30∘30^\circ30∘."

   1. Reasoning as "Conditional Synthesis": Solving a complex reasoning task is not a simple knowledge retrieval process (mapping A to B). Instead, it is a multi-condition cooperation (given A and B, derive C).

   2. In a base model, skills are fragmented: singular vector v1v_1v1​ recognizes "side lengths," v2v_2v2​ recognizes "angles", and v3v_3v3​ recognizes "area". A student who has only memorized formulas might see the side lengths (v1v_1v1​) and think of triangle inequality (u1u_1u1​), or see the angle (v2v_2v2​) and think of basic right-angle trigonometry (u2u_2u2​), and finally see area calculation(v3v_3v3​) but did not know which area formula should be used (u3u_3u3​). Without cross-talk between these features, neither path leads to the correct solution.

   3. The RL rewiring process establishes associations between these isolated points of knowledge/skills (represented by the off-diagonal terms in the MMM matrix):

    • Many-to-One Synthesis: When the model identifies both "side lengths" (v1v_1v1​) and the "included angle" (v2v_2v2​), the off-diagonal terms M3,1M_{3,1}M3,1​ and M3,2M_{3,2}M3,2​ create a logical bridge. These two signals now collectively trigger the specific output skill u3u_3u3​: the application of the Sine area formula (Area=12absin⁡Area = \frac{1}{2}ab \sinArea=21​absin)
    • Association Mechanism: The leap in reasoning capability occurs when an output decision no longer depends on a single feature match. Instead, it becomes a joint association of multiple latent premises.

Conclusion: Rewiring evolves the model from a simple "See A, Think A" mapping to a logical "See A and B, Associate C" synthesis. This transition from "point-to-point" signaling to "many-to-one" logic synthesis provides the physical foundation for the model to handle complex, multi-step reasoning tasks.

Cross-Domain Generalization

As an extension of our insight, we discovered that a reasoning update learned in the spectral space of a mathematics-focused model can be seamlessly merged into a coding-focused model. This experiment yielded a "super-model" that exhibits improved performance in both math reasoning and coding proficiency.

The Merging Algorithm

1. Extract Math Reasoning Delta: We first obtain the purified reasoning update ΔWmath∗\Delta W^*_{math}ΔWmath∗​ from a math-specialized RL model (WmathW_{math}Wmath​) using our projection method relative to the base model W0W_0W0​:

   a. ΔWmath=Wmath−W0\Delta W_{math} = W_{math} - W_0ΔWmath​=Wmath​−W0​

   b. ΔWmath∗=∑i=1ksi,math′(PUui,math′)(PVvi,math′)T\Delta W^*_{math} = \sum_{i=1}^k s_{i, math}' (P_U u_{i, math}') (P_V v_{i, math}')^TΔWmath∗​=∑i=1k​si,math′​(PU​ui,math′​)(PV​vi,math′​)T

2. Inject into Coding Manifold: We then take a coding-specialized model WcodeW_{code}Wcode​ (which is RL-finetuned from W0W_0W0​) and inject the math reasoning logic:

   a. Whybrid=Wcode+ΔWmath∗W_{hybrid} = W_{code} + \Delta W^*_{math}Whybrid​=Wcode​+ΔWmath∗​

This phenomenon suggests that "reasoning" is not a task-specific feature, but a universal spectral signature shared across different cognitive domains.

Discussion

Here we discuss the connection with existing studies:

  • In the pre-trained state, the singular vectors (ui,viu_i, v_iui​,vi​) are orthogonal, meaning different semantic "atomic skills" are largely decoupled. RL introduces non-diagonal elements in our interaction matrix MMM. These off-diagonal components act as logical bridges, allowing a signal from input feature vjv_jvj​ to trigger output feature uiu_iui​ where i≠ji \neq ji=j. This is clearly in favor of supporting that RL is incentivizing the ability of the base model.
  • By "rewiring" these previously isolated components, RL creates a more complex and flexible internal circuitry. Instead of a signal traveling through a single isolated path, it now flows through a diverse network of interconnected principal components. This directly manifests as increased activation diversity within the LLMs internal circuits.
  • 1 percent of singular component are able to recover reasoning ability, correspond to low-rankness of RL. We also observed that the magnitude of MMM is significantly smaller compared to Σ\SigmaΣ. This explains sparsity of RL: at the point of principal weights (places that WWWis large), the updates from ΔW\Delta WΔWare negiligible.
  • Comparison to SFT (Supervised FineTuning). We observed that SFT does NOT lie in the SVD of the base model. Instead, SFT significantly changes the spectrum space of the base model, which means that SFT inserts new knowledge/skills/memory into the base model. On the other hand, RL keep the existing singular component untouched, but reconnecting them. These different behavior of RL and SFT in SVD space provides a microscopic evidence for the common understanding that SFT memorizes, RL generalizes [11].

Conclusion

This study reveals the microscopic reality of RL: it does not learn new features from scratch. Instead, it "rewires" the atomic skills learned during pre-training. By optimizing the connections (the MMM matrix) between fixed geometric bases, RL enables the model to invoke its latent knowledge in a more structured and logical manner across multiple domains.
